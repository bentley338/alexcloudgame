Tutorial

Cara Main FC 25 / FC 26

01

Login Amazon Luna

Buka luna.amazon.com di Chrome. Masukkan email & password yang admin kirim lewat WhatsApp setelah pembayaran.

02

Pilih Game

Cari EA Sports FC 25 atau FC 26 di library. Klik Play — game langsung stream ke browser Anda dalam hitungan detik.

03

Save Progress

Progress otomatis tersimpan di cloud Amazon. Anda bisa keluar kapan saja dan lanjut di HP / PC lain.

04

Gunakan Controller

Pakai gamepad bluetooth (Xbox / PS4 / PS5 / generic). Untuk HP, dukungan touch controller juga tersedia.

05

Pastikan Internet Stabil

Minimal 10 Mbps untuk 1080p 60fps. WiFi 5GHz atau kabel ethernet sangat disarankan untuk FC 26.

06

Butuh Bantuan?

Hubungi admin via WhatsApp. Tim kami siap bantu troubleshooting 24/7.
